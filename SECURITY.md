
# Security Policy

## Supported Versions
We currently support the latest version of the project. Please update to the latest release before reporting a vulnerability.

## Reporting a Vulnerability
**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them by emailing **engineering@projectdavid.co.uk**.

You should receive a response within 48 hours. If the issue is confirmed, we will coordinate a fix and a disclosure with you.
